# Portion Markdown — VS Code Extension

Classification banners, portion markings, and structural validation for Markdown documents, with built-in PDF generation and live preview.

## Quick Start

```bash
cd vscode-portionmarkdown
npm install && npm run compile
```

Press **F5** in VS Code to launch the Extension Development Host. The extension activates automatically on any Markdown file.
